# ITY-Accessibility-Toolbar
You can modify it as per your own choice by adding script like this: 
<script>
    AccessibilityToolbar({
        bgColor: "rgba(0,0,0,0.6)",
        btnBgColor: "#222",
        btnTextColor: "#fff",
        btnHoverBgColor: "#fff",
        btnHoverTextColor: "#000",
        contrastBg: "#000",
        contrastText: "#ff0",
        position: "top-left",
        fontStep: 2
    });
</script>
